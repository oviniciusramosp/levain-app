# Rating avatars

Drop 3 square images here (they're clipped to circles on the site):

- avatar-1.png
- avatar-2.png
- avatar-3.png

Recommended: square (e.g. 96×96 or larger), face centered. PNG or JPG.
If a file is missing, a colored circle fallback shows instead — no breakage.
To use more/fewer, add/remove the <span> avatars in index.html (.rating__avatars)
and the matching :nth-child rules in styles.css.
